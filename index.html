A remote-controlled algorave livestream.

Undefined Control Sequence is a livestream that attempts to make live coding performances more on-the-fly. It lets the audience to control the video and sound from a 'remote controller' website and chat. While the show is going on, the audience can press buttons in the 'remote controller' or enter commands in the chat. This controls which instruments or video source will be played on the performance. The performer improvises the live set based on the audiences' choice. Commands from the website are passed on to the background video in TouchDesigner via Python, WebSocket and the Web Server DAT. Sound is performed in real-time with TidalCycles and SuperCollider.

by Jeeyoon Hyun / jeeyoonhyun.com
